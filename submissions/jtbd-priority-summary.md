# JTBD Statements

## Customer

When I call, I want to be financially prepared so that I am not surprised by the case.

## Agent

When I join a case, I want a seamless handover so that I can resolve it quickly.

When looking at cases, I want to be looking at data that makes sense, so that I
am not confused.

## Operations Manager

When Im looking at data, I want consistency across sources so that I can work efficiently.

When adopting a new workflow, I want to know it will work immediately so that my
team can do their job.

## Finance Partner

When building a case for smart-recovery, I want data and assumptions so that leadership
trusts it.
